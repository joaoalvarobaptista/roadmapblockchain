  • item 01

  • item 02

Listas de definição

Listas de definição são utilizadas para definir coisas. POr exemplo podemos usar listas de definição para escrever um material sobre matemática definindo determinado conteúdo. Para exemplo vou fazer uma lista sobre Algebra Linear definindo espaço vetorial.

Espaço vetorial
Dado um conjunto não vazio, dizemos que ele é um espaço vetorial sobre R se existem duas operações definidas com as propriedades
  • Adição
    • Comutativa
    • Associativa
    • Existencia de Elemento Neutro
    • Existência de oposto
  • Produto por escalar
    • Comutativa
    • Distributividade do escalar em relação ao vetor
    • Distributividade do vetor em relação ao escalar
    • Elemento neutro.

Lista ordenada.

Uma lista ordenada tem característica similar a lista não ordenada, mas os elementos são enumerados dando a eles uma ordenação.

  1. item 01
  2. item 02
  3. item 03
  4. item 04
